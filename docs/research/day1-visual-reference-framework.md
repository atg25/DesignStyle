## Image Quality Requirements

### For Website Use:

- **Minimum resolution:** 1920x1080px
- **Format:** JPG (photography), PNG (graphics), SVG (patterns/icons)
- **Color profile:** sRGB
- **Optimization:** Compressed but high quality
- **Copyright:** Must have rights or public domain

### For Reference:

- Highest resolution available
- Multiple angles when possible
- Detail shots plus context shots
- Before/after (restorations)
- Original catalog images when available

---

## Copyright & Usage Notes

### Public Domain Sources:

- Pre-1928 publications (US)
- Government archives
- Explicit public domain releases
- Creative Commons CC0

### Fair Use Considerations:

- Educational purpose
- Non-commercial (academic project)
- Transformative use
- Limited portions
- Credit sources

### Best Practices:

- Document every source
- Credit photographers (Julius Shulman, etc.)
- Link to museums/archives
- Respect designer legacy
- When in doubt, contact rightsholder
