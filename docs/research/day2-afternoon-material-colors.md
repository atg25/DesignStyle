## Part 1: Wood Tones (The Foundation of MCM)

### Teak - THE MCM Wood

**Why Teak Was King:**

- **Natural oils**: Water and rot resistant (no treatment needed)
- **Stability**: Doesn't warp or split
- **Workability**: Easy to carve and shape
- **Color**: Warm honey-brown with golden undertones
- **Aging**: Beautiful silver-gray patina if left outdoors
- **Status**: Exotic import = luxury (Burma, Thailand, Indonesia)

**Teak Color Variations:**

```css
/* Fresh/New Teak */
--teak-light: #d2a679; /* Light honey */
--teak-medium: #b8860b; /* Golden brown - MOST COMMON */
--teak-dark: #8b4513; /* Rich saddle brown */

/* Aged Teak */
--teak-aged-warm: #a67b5b; /* Warm aged patina */
--teak-aged-cool: #9c8273; /* Cooler aged tone */

/* Oiled/Finished Teak */
--teak-oiled: #c19a6b; /* Teak oil enhances gold */
--teak-waxed: #b8956a; /* Wax gives matte finish */
```

**Grain Characteristics:**

- Straight grain (some wavy)
- Medium to coarse texture
- Visible growth rings
- Occasional dark streaks
- Natural luster (even unfinished)

**Famous Teak Pieces:**

- Danish credenzas (sideboards)
- Hans Wegner chairs
- Børge Mogensen furniture
- Teak + rosewood combinations

**Digital Representation:**

- Show wood grain texture
- Subtle gradients (not flat color)
- Include darker streaks
- Highlight natural luster

### Walnut - American Classic

**Why Walnut Dominated American MCM:**

- **Domestic availability**: Grows in USA (cheaper than imported teak)
- **Rich color**: Darker, more dramatic than teak
- **Strength**: Extremely durable
- **Workability**: Excellent for carving, bending
- **Prestige**: Associated with fine furniture for centuries
- **Contrast**: Beautiful with lighter colors

**Walnut Color Spectrum:**

```css
/* American Black Walnut */
--walnut-sapwood: #8b7355; /* Lighter outer wood */
--walnut-heartwood: #5c4033; /* Classic dark brown - MOST COMMON */
--walnut-dark: #3d2817; /* Nearly chocolate */

/* Color Variations */
--walnut-purple-cast: #4a3c31; /* Purplish undertone (common) */
--walnut-gray-cast: #564a3d; /* Grayish undertone */
--walnut-warm: #654321; /* Warmer reddish brown */

/* Finished Walnut */
--walnut-oiled: #6b4423; /* Oil darkens slightly */
--walnut-lacquered: #5a3825; /* Lacquer deepens color */
```

**Grain Characteristics:**

- Usually straight (can be wavy)
- Medium texture
- Beautiful figure (burl, crotch wood)
- Dark streaks common
- Rich, chocolatey

**Famous Walnut Pieces:**

- Eames Lounge Chair (original)
- George Nelson credenzas
- Knoll furniture
- American studio furniture

**Pairing Guidelines:**

- Works with ANY color
- Best with: White, cream, gray, orange, teal
- Grounds bright colors
- Sophisticated alternative to teak

### Rosewood - Luxury Material

**Why Rosewood Meant Luxury:**

- **Rarity**: Exotic (Brazil, India)
- **Cost**: Expensive even then
- **Beauty**: Dramatic grain, purple-brown color
- **Prestige**: Used only in finest pieces
- **Fragrance**: Pleasant scent when worked

**Rosewood Colors:**

```css
/* Brazilian Rosewood */
--rosewood-light: #8b4513; /* Medium brown */
--rosewood-typical: #65000b; /* Deep burgundy-brown - ICONIC */
--rosewood-dark: #3b0b0b; /* Nearly black with red */

/* Grain Variations */
--rosewood-purple: #5c2e2e; /* Purple undertones */
--rosewood-golden: #8b6914; /* Golden streaks */
--rosewood-black: #2b1b17; /* Black streaks */
```

**Grain Characteristics:**

- Highly figured (dramatic patterns)
- Interlocked grain
- Oily appearance
- Natural luster
- Dark streaks and veins

**Famous Rosewood Pieces:**

- Eames Lounge Chair (luxury version)
- Danish modern cabinets
- Mies van der Rohe Barcelona pieces
- Piano finishes

**Modern Status:**

- Now restricted (CITES protected)
- Vintage rosewood highly valuable
- Reproductions use other woods

### Birch/Beech - Scandinavian Staples

**Why Light Woods in Scandinavia:**

- **Local availability**: Native to Nordic countries
- **Light climate**: Maximizes light in dark winters
- **Affordability**: Common wood = accessible design
- **Workability**: Excellent for bending (Aalto)
- **Clean aesthetic**: Matches Scandinavian minimalism

**Birch Colors:**

```css
/* White/Yellow Birch */
--birch-light: #f3e5ab; /* Nearly blonde */
--birch-typical: #e8d6a1; /* Cream yellow - MOST COMMON */
--birch-reddish: #d2b48c; /* Slight red tinge */

/* Finnish Birch (Prized) */
--finnish-birch: #fae7b5; /* Very light, golden */
```

**Beech Colors:**

```css
/* European Beech */
--beech-white: #f5deb3; /* Whitish cream */
--beech-typical: #c19a6b; /* Light tan - MOST COMMON */
--beech-steamed: #d2a679; /* Steaming adds pink tone */
```

**Famous Light Wood Pieces:**

- Alvar Aalto bent plywood chairs
- Stool 60 (Aalto) - stacks
- Series 7 Chair (Jacobsen)
- Bruno Mathsson lounge chairs

**Characteristics:**

- Fine, even grain
- Light, almost white
- Ages to warmer tone
- Bends easily when steamed

### Oak - Traditional American

**Why Oak in American MCM:**

- **Tradition**: Historic American wood
- **Availability**: Grows throughout USA
- **Strength**: Extremely durable
- **Cost**: Affordable
- **Recognition**: Familiar to consumers

**Oak Colors:**

```css
/* White Oak */
--oak-white: #e8d5b7; /* Pale tan */
--oak-white-typical: #d4c4a8; /* Light brown - MOST COMMON */

/* Red Oak */
--oak-red: #b87860; /* Pinkish brown */
--oak-red-typical: #a67b5b; /* Warm brown */

/* Quartersawn Oak */
--oak-quartersawn: #c2b280; /* Shows ray fleck pattern */
```

**Grain Characteristics:**

- Very prominent grain
- Open pore structure
- Ray fleck (quartersawn)
- Rustic appearance

**Usage:**

- Less common than teak/walnut in high design
- More in mass-market furniture
- Late period (1965-1970) revival
- DIY and craft movement

## Part 2: Upholstery & Fabric Colors

### Leather

**Why Leather in MCM:**

- **Luxury**: Premium material
- **Durability**: Lasts decades
- **Patina**: Ages beautifully
- **Comfort**: Breathable, supple
- **Modern**: Clean, not fussy

**Leather Colors:**

```css
/* Classic MCM Leather */
--leather-cognac: #9b4c3a; /* Classic brown - MOST ICONIC */
--leather-black: #1c1c1c; /* Black (second most common) */
--leather-chocolate: #3d2817; /* Dark brown */
--leather-caramel: #c68e17; /* Light tan */
--leather-oxblood: #4a0404; /* Burgundy (rare, luxe) */

/* Aged Leather */
--leather-aged-cognac: #8b4513; /* Darkens with age */
--leather-worn-black: #2c2c2c; /* Softens to charcoal */
```

**Texture Notes:**

- Top-grain leather (highest quality)
- Natural markings expected
- Develops patina over time
- Can be smooth or pebbled

**Famous Leather Pieces:**

- Eames Lounge Chair (cognac or black)
- Barcelona Chair (black or cognac)
- Womb Chair (optional leather)

---

### Wool Fabrics

**Why Wool Dominated:**

- **Durability**: Tough, resilient
- **Comfort**: Warm but breathable
- **Dye-ability**: Takes color well
- **Texture**: Adds visual interest
- **Quality**: Premium feel

**Wool Fabric Colors:**

```css
/* Solid Wool Colors */
--wool-charcoal: #36454f; /* Dark gray */
--wool-oatmeal: #e8dcc4; /* Neutral beige */
--wool-rust: #b7410e; /* Warm terracotta */
--wool-orange: #ff6b35; /* Vibrant orange */
--wool-teal: #1fb7c8; /* Bright teal */
--wool-mustard: #c19a6b; /* Warm yellow-brown */
--wool-olive: #808000; /* Muted green */
```

**Texture Variations:**

- **Boucle**: Nubby, looped (famous MCM texture)
- **Herringbone**: Diagonal weave pattern
- **Tweed**: Flecked, multi-color
- **Felt**: Dense, smooth
- **Mohair**: Fuzzy, luxurious

**Famous Wool Applications:**

- Womb Chair (Alexander Girard fabrics)
- Knoll furniture
- Herman Miller upholstery options

### Naugahyde (Vinyl)

**Why Synthetic Leather:**

- **Cost**: Much cheaper than real leather
- **Easy Care**: Wipe clean
- **Colors**: Any color possible
- **Modern**: Space-age material
- **Consistency**: No natural variations

**Naugahyde Colors:**

```css
/* Popular Vinyl Colors */
--vinyl-black: #0a0a0a; /* Black (most common) */
--vinyl-white: #f5f5f5; /* White (modern, risky) */
--vinyl-red: #c1272d; /* Bright red */
--vinyl-orange: #ff4500; /* Orange-red */
--vinyl-turquoise: #40e0d0; /* Bright aqua */
--vinyl-yellow: #ffd700; /* Golden yellow */
```

**Characteristics:**

- Shiny surface
- Smooth texture
- Can crack with age
- Heat-sensitive
- Atomic age appeal

**Usage:**

- Diner booths
- Commercial seating
- Budget furniture
- Children's furniture

## Part 3: Hard Surface Materials

### Formica Laminates

**Why Formica Revolutionized Interiors:**

- **Durability**: Heat and scratch resistant
- **Easy Cleaning**: Wipe with damp cloth
- **Affordability**: Cheap to produce
- **Colors**: Endless possibilities
- **Patterns**: Could print anything

**Solid Formica Colors:**

```css
/* Popular Solid Colors */
--formica-white: #fafafa; /* Clean white */
--formica-yellow: #ffed4e; /* Bright sunny */
--formica-turquoise: #00ced1; /* Vibrant teal */
--formica-pink: #ffb6c1; /* Bubblegum pink */
--formica-green: #90ee90; /* Mint green */
--formica-gray: #d3d3d3; /* Neutral gray */
```

**Pattern Categories:**

**1. Boomerang Pattern:**

```css
/* Background colors for boomerang patterns */
--boomerang-bg-cream: #faf0e6;
--boomerang-bg-gray: #c0c0c0;
--boomerang-bg-black: #2c2c2c;

/* Boomerang shape colors (typically 3-4 colors) */
--boomerang-orange: #ff6b35;
--boomerang-teal: #1fb7c8;
--boomerang-yellow: #ffd700;
--boomerang-black: #000000;
```

**2. Atomic/Starburst:**

```css
/* Starburst typically on pale background */
--atomic-bg: #fffaf0; /* Cream/ivory */
--atomic-accent1: #ff6347; /* Tomato red */
--atomic-accent2: #4169e1; /* Royal blue */
--atomic-accent3: #ffd700; /* Gold */
```

**3. Speckled/Confetti:**

```css
/* Base color with flecks */
--speckle-base: #f5f5dc; /* Beige base */
--speckle-dots: Various bright colors in pattern;
```

**Famous Patterns:**

- "Skylark" (boomerang)
- "Capri" (atomic)
- "Confetti" (speckled)
- "Woodgrain" (faux wood - ironic!)

**Applications:**

- Kitchen countertops (main use)
- Bathroom vanities
- Table tops
- Cabinet doors
- Wall panels

### Terrazzo Flooring

**What is Terrazzo:**

- Chips of marble, quartz, granite, or glass
- Set in concrete or epoxy
- Polished to smooth surface
- Creates speckled appearance

**Terrazzo Colors:**

```css
/* Base Colors */
--terrazzo-base-white: #f8f8ff; /* White cement */
--terrazzo-base-gray: #808080; /* Gray cement */
--terrazzo-base-red: #8b4513; /* Red cement (rare) */

/* Chip Colors (mixed in) */
--terrazzo-chip-white: #ffffff; /* White marble */
--terrazzo-chip-black: #000000; /* Black marble */
--terrazzo-chip-pink: #ffb6c1; /* Pink marble */
--terrazzo-chip-green: #90ee90; /* Green marble */
--terrazzo-chip-gold: #ffd700; /* Brass dividers */
```

**Characteristics:**

- Durable, permanent
- Cool to touch
- Reflects light
- Mid-century schools and commercial buildings
- Residential use (luxury)

---

### Linoleum Flooring

**Why Linoleum (Not Vinyl):**

- Natural material (linseed oil)
- Eco-friendly (before it was trendy)
- Resilient, comfortable underfoot
- Available in many colors/patterns

**Linoleum Colors:**

```css
/* Solid Colors */
--lino-red: #C1272D;            /* Classic red */
--lino-black: #1C1C1C;          /* Black (checkerboard) */
--lino-white: #F5F5F5;          /* White (checkerboard) */
--lino-green: #228B22;          /* Forest green */
--lino-yellow: #FFD700;         /* Bright yellow */
--lino-gray: #808080;           /* Neutral gray */

/* Pattern Styles */
--lino-marbled: Mixed colors swirled together
--lino-speckled: Flecked with multiple colors
--lino-inlaid: Geometric patterns cut and fitted
```

**Famous Pattern:**

- Black and white checkerboard (THE MCM kitchen floor)
- Marbled green (hospitals, schools)
- Inlaid geometric (high-end residential)

## Part 4: Metal Finishes

### Chrome - THE MCM Metal

**Why Chrome Dominated:**

- **Modern**: Shiny, technological
- **Reflective**: Bounces light around
- **Durable**: Rust-resistant
- **Versatile**: Works with any color
- **Affordable**: Plating process cheap

**Chrome Colors:**

```css
/* Chrome isn't a "color" but reflects environment */
--chrome-highlight: #E8E8E8;    /* Bright reflections */
--chrome-midtone: #C0C0C0;      /* Medium silver */
--chrome-shadow: #808080;       /* Darker areas */

/* Digital representation needs gradient: */
linear-gradient(135deg, #E8E8E8 0%, #C0C0C0 50%, #808080 100%)
```

**Surface Finishes:**

- **Polished Chrome**: Mirror-like (most common)
- **Brushed Chrome**: Satin, less reflective
- **Chrome-plated**: Over steel base

**Famous Chrome Pieces:**

- Eames Aluminum Group chairs
- Barcelona Chair base
- Tulip Chair base (later versions)
- Hairpin legs
- Bertoia Wire Chairs

**Digital Tips:**

- Show reflections
- Add highlights
- Include subtle gradients
- Consider environmental lighting

---

### Brass - Warm Accent

**Why Brass (Less Common):**

- **Warmth**: Golden tone
- **Luxury**: More expensive than chrome
- **Patina**: Ages to antique look
- **Traditional**: Nod to past

**Brass Colors:**

```css
/* Brass Variations */
--brass-polished: #b5a642; /* Shiny yellow-gold */
--brass-satin: #aa9a5a; /* Brushed, matte */
--brass-aged: #8b7355; /* Oxidized, brownish */
--brass-antique: #6b5d57; /* Heavily aged */
```

**Usage:**

- Light fixtures (common)
- Hardware (drawer pulls)
- Decorative accents
- Table legs (rare)

**Late Period (1965-1970):**

- Brass revival
- Warmer than chrome
- Bohemian influence

- **Craft**: Handmade feel

**Copper Colors:**
